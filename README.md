# Pocket Browser PDF Book Reader.

## Personal PDF viewer:

- Fetch books
- Open book
- Control typography/zoom
- Resume reading where you left via `localStorage` (later via could)
- Dark/Light modes
- Search in PDF

## Performance Reqs:

- < 1s load time 4G, lazy loading
- Offline friendly
